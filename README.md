**🎮 How to Play**

1.	Clone or download this repository to your local system. 
2.	Make sure Python 3.x is installed on your computer.
3.	Open a terminal or command prompt in the project directory.
4.	Run the game using: **(python main.py)**
5.	Enter your name when prompted to begin the quiz.
6.	Read each multiple-choice Python question carefully.
7.	Type the correct option (A, B, C, or D) and press Enter.
8.	If your answer is correct, you move on to the next question and increase your score.
9.	If your answer is incorrect, the game ends and your final score is displayed.
10.	Complete all questions correctly to become the **Python Brilliant Quiz Champion!**


**✨ Features**

1.	Interactive command-line quiz game 
2.	Multiple-choice Python programming questions
3.	Instant feedback for each answer
4.	Score tracking throughout the game
5.	Beginner-friendly and interview preparation focused
6.	Easy to customize by adding new questions and difficulty levels
